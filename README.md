# Meme Generator

This is a web application built using React that allows users to create memes by adding captions to pre-existing images. This application uses the Meme Generator API to fetch images and the captions submitted by users.

## Getting Started

To use this application, you can either clone this repository or download the source code. Once you have the source code, navigate to the root directory of the project in your terminal and run `npm install` to install all the necessary dependencies.

Next, you can start the application by running `npm start`. This will start a development server on your local machine, which you can access by navigating to `http://localhost:3000` in your web browser.

## Usage

To use the meme generator, follow these steps:

1. Choose a template. Press "Get a new meme" button to get a randomized meme from a library of over 100 memes.
2. Add text. When a desired meme has been chosen, enter the top text and bottom text desired to display on the meme.
3. Save and share. Once you are happy with the meme you have created, you can share through social apps, or download the meme to your device.
